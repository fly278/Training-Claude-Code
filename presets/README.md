# 预设 / Presets

> 选择适合你的预设，一键应用。

---

## 如何使用

```bash
# 选择一个预设，复制为你的 CLAUDE.md：
cp presets/minimal.md ~/.claude/CLAUDE.md    # 最简洁
cp presets/standard.md ~/.claude/CLAUDE.md   # 推荐
cp presets/power.md ~/.claude/CLAUDE.md      # 最强大
```

## 自定义

预设不是最终答案。用了一段时间后，根据实际体验增删规则：

1. 哪些规则经常被触发但效果好？→ 保留
2. 哪些规则经常被忽略？→ 删除
3. 哪些场景没有覆盖？→ 添加
